# Engagement Types

The Commercial Strategy Platform uses exactly three engagement types:

1. Executive Hire
2. Consulting Engagement
3. Agency Engagement

The Opportunity Model stays consistent across all three. The engagement type determines audience framing, required inputs, and deliverables.

## How Engagement Types Work

Every engagement starts with an Opportunity Intake, Intake Engine, Discovery Engine, Commercial Intelligence, and an Opportunity Model.

```text
Opportunity Intake
Intake Engine
Discovery Engine
Commercial Intelligence
Opportunity Model
Output Generator / Renderer
Deliverable
```

The model should continue to use the canonical Commercial Strategy sections when rendered as a website or proposal-style output:

```text
FOUNDATION
01. KPI Foundation
02. Segmentation
03. Total Addressable Market
04. Audience Architecture

EXECUTION
05. Prospect Funnel
06. Signal Intelligence
07. Commercial Activation

MEASUREMENT & SYSTEM
08. Sales Motion
09. M+S Alignment
10. Return to Dashboard
```

## Engagement Type Summary

| Engagement type | Primary audience | Typical deliverables |
| --- | --- | --- |
| Executive Hire | Hiring leader, executive team, recruiter, board-adjacent stakeholders | Strategy page, interview prep, cover letter, LinkedIn/recruiter outreach |
| Consulting Engagement | Client executive sponsor, functional leaders, implementation stakeholders | Proposal, diagnostic brief, executive dashboard, roadmap, workshop narrative |
| Agency Engagement | Client marketing/revenue leader, agency buyer, account stakeholders | Campaign strategy, account plan, activation proposal, performance dashboard, sales enablement narrative |

## Non-Negotiables

- Do not add additional engagement types without explicitly updating this contract.
- Do not treat deliverables as engagement types.
- Do not change the Opportunity Model shape for each engagement type.
- Do adapt audience framing, input requirements, and deliverables by engagement type.
- Do preserve the current Commercial Strategy page navigation for website/proposal-style Renderers.
