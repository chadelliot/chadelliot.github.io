# AboutChad React SPA (root domain)

Single-page React app that loads once and navigates between sections without refresh:
- About (Hero + Marquee + Pillars)
- Approach (QXO Story + 4 tabs)
- Career (Timeline)
- Skills (Expertise + Education + Certifications)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to aboutchad.com
This project is configured for a *root domain* (`base: "/"`).
Deploy the `dist/` folder to your hosting target.
